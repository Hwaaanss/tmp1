# 프로젝트 청사진: 로또 번호 추첨기

## 프로젝트 개요
이 프로젝트는 웹 표준(HTML, CSS, JavaScript)과 웹 컴포넌트를 사용하여 구축된 현대적인 '로또 번호 추첨기' 애플리케이션입니다. 사용자에게 시각적으로 즐거운 경험을 제공하며, 모바일 대응이 가능한 반응형 디자인을 갖추고 있습니다.

## 주요 기능 및 디자인
- **웹 컴포넌트 (`lotto-ball`):** 개별 로또 번호를 캡슐화된 섀도 DOM(Shadow DOM) 스타일로 표시합니다.
- **애니메이션 효과:** 번호 추첨 시 순차적으로 나타나는 애니메이션 효과를 제공합니다.
- **반응형 레이아웃:** 모바일과 데스크톱 모두에서 최적화된 화면을 보여줍니다.
- **세련된 UI:** 그림자 효과, 그라디언트, 글로우 버튼 등을 사용하여 현대적인 느낌을 줍니다.

## 업데이트 기록
### 글씨체 변경 (고딕체 적용)
- **목적:** 애플리케이션의 가독성을 높이고 현대적인 고딕체 느낌을 제공.
- **변경 사항:**
    1. `index.html`: Google Fonts의 'Nanum Gothic' 폰트 임포트 코드 추가.
    2. `style.css`: 전역 폰트를 'Nanum Gothic' 및 시스템 고딕체 폴백으로 변경.
    3. `main.js`: `lotto-ball` 웹 컴포넌트의 섀도 DOM 내부 스타일에도 'Nanum Gothic' 적용.
